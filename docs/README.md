# Docs 目录说明

本目录包含项目开发过程中的各类文档，各子目录用途如下：

## 目录结构

| 目录 | 用途 |
|------|------|
| `usage/` | **用户使用文档** — 面向最终用户的功能说明与使用指南 |
| `prd/` | 产品需求文档 — 项目初期的需求概览与定义 |
| `planning/` | 规划文档 — 由不同 AI 工具生成的产品规划、技术设计与任务拆解记录 |
| `features/` | 功能文档 — 各功能特性的详细说明与开发记录 |
| `cli/` | CLI 命令 — 命令行接口的设计与实现过程记录 |
| `tui/` | TUI 看板 — 终端看板界面的设计与实现过程记录 |
| `agent/` | AI Agent 集成 — Claude Code、Codex 等 Agent 的接入与执行引擎过程记录 |
| `bot/` | Bot 集成 — 飞书/微信 Bot 的消息接收、解析与处理过程记录 |
| `notification/` | 通知系统 — 多渠道任务通知（飞书、微信等）的设计与实现过程记录 |
| `sync/` | 数据同步 — 飞书多维表格双向同步的设计与实现过程记录 |
| `storage/` | 存储层 — 本地文件存储（Markdown + Frontmatter）与 SQLite 的设计过程记录 |

> ⚠️ 除 `usage/` 为最终面向用户的使用文档外，其余目录均为开发过程中的过程文件记录，仅作归档用途，不代表最终产品状态。
