# Local Task Center UI Spec 总览

## 目标

定义 Task Center 在三类 UI 形态中的职责边界、模块拆分与实施顺序。

## UI 形态

- CLI/TUI：P0 主界面
- Web：P1 观察与管理界面
- Desktop：P2 本地增强界面

## 共用 UI 模块

- Capture Entry
- Task Inbox
- Workflow Board
- Dispatch Panel
- Execution Monitor
- Review Queue
- Dashboard Views
- Notification Surface
